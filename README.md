# SteamworksConnectionLinux

Linux implementation of SteamworksConnection

# Commands
## Installation
```./install.sh```
## Uninstallation
```./uninstall.sh```
## Building
```./build.sh```
Uses g++ to compile source files
## Generating h/cc files
```./makeprotos.sh```

## Running
```./swc```

# Dependencies
Google Protocol Buffers v2.5.0
g++
